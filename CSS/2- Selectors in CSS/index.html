Selectors in HTML & CSS

Element Selector

The element selector selects HTML elements based on the element name.

Id Selector

The id selector uses the id attribute of am HTML element to select a specific element. The Id of an element is unique within a page, so the id selector is used to select one unique element!

Class Selector

The Class Selector selects HTML elements with a specific class attribute. To select elements with a specific class, write a period(.) character, followed by the class name
